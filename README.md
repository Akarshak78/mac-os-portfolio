# 🍎 macOS Portfolio

A modern **macOS-inspired portfolio** built with **React.js**, recreating the macOS desktop experience in the browser with draggable windows, a responsive dock, an interactive terminal, and multiple desktop applications.

![React](https://img.shields.io/badge/React-19-61DAFB?logo=react)
![Vite](https://img.shields.io/badge/Vite-7-646CFF?logo=vite)
![SCSS](https://img.shields.io/badge/SCSS-C76494?logo=sass)
![License](https://img.shields.io/badge/Status-Active-success)

---

## ✨ Features

- 🖥️ macOS-inspired desktop interface
- 🪟 Draggable and resizable windows
- 🎯 Active window focus with dynamic z-index
- 📂 GitHub Projects window
- 📄 Resume Viewer
- 🎵 Spotify Embed
- 💻 Interactive Terminal
- 📝 Notes Application
- 🕒 Live Menu Bar Clock
- ⚡ Dock with hover magnification
- 🎨 Modern glassmorphism UI

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/Akarshak78/mac-os-portfolio.git
```

Install dependencies

```bash
npm install
```

Start development server

```bash
npm run dev
```
